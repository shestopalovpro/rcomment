# Comment Rating Plugin

## Description
A WordPress plugin that adds upvote/downvote functionality to comments with a modern, minimalist design.

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The voting buttons will automatically appear below each comment

## Features
- Modern SVG icons for voting
- AJAX-powered voting (no page reload)
- Prevents duplicate votes
- Works with both logged-in users and guests
- Responsive design
- Dark mode support
- Universal design that works with all themes
